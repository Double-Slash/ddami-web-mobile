# DDAMI - web (mobile)

### Development Stack
- React.js

### Commit Message Convention

```
Feat : 새로운 기능 추가
Fix : 버그 수정
Docs : 문서 수정
Style : 코드 포맷팅, 세미콜론 누락, 코드 변경이 없는 경우
Refactor : 코드 리펙토링
Chore : 기타 작업
Branch 이름은 기능별로 생성
```

### Page URI

- `/` : 메인
- `/join` : 회원가입
- `/login` : 로그인
- `/search` : 검색
- `/workplace` 작업실 관련
    - `/my` : 내 작업실
    - `/:id` : 아이디가 id인 사용자의 작업실
    - `/work/:workId` : workId의 작업물 상세 페이지
    - `/write` : 작업실 글 작성하기
- 햄버거 관련 페이지
    - `/like` : 좋아요한 작품
    - `/purchase` : 판구매 내역
    - `/subscribe` : 찜한 목록
    - `/setting` : 설정
